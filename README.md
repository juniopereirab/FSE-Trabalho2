## Como executar

Entrando na pasta raíz do projeto em: https://github.com/juniopereirab/FSE-Trabalho2

Execute os comandos:

```
cd Trabalho
make
bin/bin
```

Os comando `make` irá compilar o código e o `bin/bin` irá executá-lo.
